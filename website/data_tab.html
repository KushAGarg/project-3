We looked at over 65,000 reported UFO sightings between the years of 1910-2014.

Our Data

The original data came as a .csv file from Kaggle that contained over 70,000 UFO sightings between 1910 and 2014. It contained the following columns: datetime, city, state, country, shape, duration (seconds), duration (hours/min), comments, date posted, latitude,longitude. We appended an ID column and lightly cleaned the data. The dataset had obvious limitations, but it was relatively easy to manipulate.
